### Code Samples

```cpp

x = cndtn ? val1 : val2

// both are same

if(cndtn){
	x=val1;
}
else{
	x=val2;
}

// flips 0 to 1 
// also 1 to 0
// for all i
A[i]=A[i]==0? 1 : 0
```

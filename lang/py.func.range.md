### range() vs xrange

- Why Better to Use range
  - xrange is deprecated in python3 
  - all list methods can be used in range
- Difference [GFG](https://www.geeksforgeeks.org/range-vs-xrange-python/)
  - Return Type 
    - range() > range type list iterable 
    - xrange() > xrange object
  - Memory 
    - range() takes more
  - Speed 
    - xrange() is faster

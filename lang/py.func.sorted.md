- Summary 
	- returns a sorted list of specified iterable object 
- Syntax
	- `sorted(iterable, ?key=None, ?reverse=False)`
	- parameters
		- iterable 
			- sequence 
				- strings
				- lists
				- tuples
			- collections
				- set
				- dictionary
				- frozen set
			- any other iterable
		- reverse (optional)
			- default 
				- False
		- key (optional)
			- default 
				- None
			- A function that serves as key for sort comparison
